How To Play ❓

When the game starts click on the right answer to score a point, clicking on the wrong answer will not give you any points. There is a timer located at the top of the game section and its speed varies on the difficulty level you are playing on. If the timer elapses you fail that game round. Note that divisions with decimals are rounded up to the nearest whole number to make the game look clean. Failing all set rounds will conclude the game.

Difficulty

There are 3 difficulties in this game: Easy, Medium and Hard. Each difficulty affects the timer speed from 1.5 seconds (Hard) to 4.5 seconds (Easy). Each second determines the rate of change of the timer numbers ,that is in Easy mode it will take 1.5 seconds to count from 3 to 2 then to 1 making Easy mode 4.5 seconds to finish 1 round.

Shortcuts

There are various shortcuts in the game but the list below shows the most important for ease and accessibility, hovering on the listed items will show more info about them:

  • Alt + 1/ 2/ 3 : Will select option 1/ 2/ 3 respectively.
  • Esc : Can be used to close some tabs.
  • Alt + S : Will open settings tab.
  • Alt + Q : Will make a quick game.
  • Alt + V : Will reduce game volume.
  • Alt + Shift + V : Will increase game volume.
  • Alt + H : Will open How To Play Tab.
